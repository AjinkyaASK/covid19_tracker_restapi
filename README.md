## Key Features

* Live Data for Covid-19 Pandemic
* Pull Down to Refresh
* Single Page UI for simplicity
* Uses API nCoV2019 2.0.0 for the data
* Provider
* Dark/Light mode
* Cross platform (iOS and Android)
